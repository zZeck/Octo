Only gchip from the chromatophore repo seems to work in emu48
Load object and select the binary
press the alpha key and number keys to enter a string
press enter to put string on stack
STO to store the gchar binary using name of string entered.
Drag chip8 program onto emu48 or use load object to load chip8 program.
Will show first few bytes as string on stack.
VAR to show available programs
press the F1-F6 key corresponding to gchip.
gchip will run the program.
